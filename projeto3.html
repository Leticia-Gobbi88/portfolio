HuggingFace LLM's Test

O objetivo desse projeto é utilizar um modelo de linguagem de grande escala (LLM) para gerar textos, responder perguntas, realizar tradução, análise de sentimentos e experimentos com técnicas de prompt engineering. O projeto explora como modelos de IA, como o Phi-3-mini-4k-instruct da Microsoft, podem ser aplicados a diferentes tarefas de processamento de linguagem natural (NLP)..

Ferramentas e Tecnologias Utilizadas

• Linguagem de Programação: Python

• Bibliotecas: torch → Para manipulação de tensores e uso da GPU; transformers (Hugging Face) → Para carregar e utilizar modelos de linguagem pré-treinados

• Modelo de IA: microsoft/Phi-3-mini-4k-instruct (um modelo de geração de texto da Microsoft)

• Hardware: GPU (ex.: P100) para acelerar o processamento Acesso à Internet para baixar modelos e configurações

Processo de Aprendizado e Desenvolvimento

Esse projeto demonstra como modelos de IA podem ser usados para diversas tarefas de NLP, explorando desde a geração de texto até tradução e análise de sentimentos. A principal lição aprendida é a importância do ajuste de prompts e hiperparâmetros para obter melhores respostas. Modelos menores têm mais dificuldade em seguir instruções complexas, e o uso de GPUs acelera o processamento.
